# Mens-T-Shirt-Brand-Analytics-Dashboard-Azure-Power-BI

## 📌 Project Overview

This project demonstrates an end-to-end business intelligence solution built using Azure and Power BI to analyze men's fashion retail data.

The dashboard provides insights into brand performance, sales trends, product profitability, discount strategies, and product distribution across different brands. The objective is to help retail businesses identify high-performing brands, optimize pricing strategies, and improve profit margins through data-driven decision-making.

---

# 🎯 Business Problem

Fashion retailers often manage products from multiple brands with varying pricing, discount structures, and profitability levels.

Without proper analysis, businesses may struggle to:

* Identify top-performing brands
* Evaluate discount effectiveness
* Monitor profit margins
* Compare sales performance across brands
* Understand product distribution
* Optimize pricing strategies

This dashboard addresses these challenges by providing interactive analytics that enable stakeholders to monitor brand performance and business profitability.

# 📂 Dataset Overview

The dataset contains product-level information for men's t-shirts.

### Key Fields

| Field            | Description            |
| ---------------- | ---------------------- |
| Brand            | Product Brand          |
| Title            | Product Name           |
| Sale Price       | Selling Price          |
| Discount (%)     | Discount Applied       |
| Profit (%)       | Product Profit Margin  |
| Product Category | Product Classification |

# ⚙️ Data Preparation Process

## Step 1: Data Ingestion

Retail product data was imported into Azure for centralized storage and processing.

## Step 2: Data Cleaning

Data transformation activities included:

* Removing duplicate records
* Handling missing values
* Standardizing brand names
* Validating pricing fields
* Formatting numerical measures
* Preparing reporting-ready datasets

## Step 3: Data Validation

The dataset was validated to ensure:

* Accurate pricing calculations
* Consistent profit metrics
* Reliable brand categorization
* Data completeness

## Step 4: Power BI Integration

Power BI was connected to the transformed dataset for reporting and visualization.

## 2️⃣ Brand Performance Analysis

This page provides detailed insights into sales, discounts, profitability, and product distribution.

### Visualizations Included

#### Brand vs Discount Percentage

Analyzes the total discount percentage applied across brands.

**Insight**

* Helps evaluate promotional strategies.
* Identifies brands relying heavily on discounts.

---

#### Product Count by Brand

Measures the number of products available under each brand.

**Insight**

* Highlights brands with the largest product portfolios.
* Supports inventory planning.

---

#### Sales Performance by Brand

Compares total sales revenue generated by each brand.

**Insight**

* Identifies top revenue-generating brands.
* Supports strategic brand investment decisions.

---

#### Average Profit Percentage by Brand

Shows average profitability across brands.

**Insight**

* Identifies brands with stronger margins.
* Helps optimize product mix strategies.

---

#### Total Profit Contribution by Brand

Analyzes overall profit contribution from each brand.

**Insight**

* Highlights the most profitable brands.
* Supports profitability-focused decision-making.
